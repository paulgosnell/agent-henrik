## ðŸ—ºï¸ Managing Destinations

### Viewing Destinations

1. Click **"Destinations"** in the sidebar
2. You'll see a list of all destinations
3. Use **Search** to find specific destinations
4. Use **Filters** to show only certain categories or published/draft items

### Adding a New Destination

1. Click the **"+ Add New Destination"** button
2. Fill in the form:

**Required Fields:**
- **Title** - Destination name (e.g., "Stockholm")
- **Slug** - Auto-generated URL-friendly version
- **Description** - 2-3 sentences about the destination
- **Image** - Upload a hero image (see below)
- **Latitude & Longitude** - Click on the map!
- **Category** - Choose from dropdown (City, Seaside, Province, etc.)
- **Themes** - Check all that apply
- **Seasons** - Check best times to visit

**How to Set Location:**
1. Look at the interactive map in the form
2. **Click** on the map where the destination is
3. A marker will appear
4. Latitude and Longitude fields update automatically
5. You can also type coordinates if you prefer

**How to Upload Image:**
1. Click the image upload area (or drag & drop)
2. Choose a high-quality image (1920Ã—1280 recommended)
3. Image preview appears
4. File uploads when you save

3. Click **"Save & Publish"** to make it live
4. Or click **"Save as Draft"** to work on it later

### Editing a Destination

1. Find the destination in the list
2. Click anywhere on that row
3. Form opens with current information
4. Make your changes
5. Click **"Update Destination"**

### Deleting a Destination

1. Find the destination
2. Click the **red trash icon**
3. Confirm deletion
4. Destination is permanently removed

## ðŸ“ Creating Blog Posts

### Adding a New Post

1. Click **"Blog Posts"** in sidebar
2. Click **"+ Add New Post"**
3. Fill in the form:

**Title & Basics:**
- **Title** - Your blog post headline
- **Slug** - Auto-generates, but you can edit it
- **Excerpt** - Short summary (shows in previews)
- **Author** - Your name (defaults to "LIV Team")

**Content:**
- Use the **rich text editor** to write your article
- Toolbar buttons let you:
  - **B** - Make text bold
  - **I** - Make text italic
  - **Link** - Add links
  - **Image** - Insert images
  - **Lists** - Create bullet points
  - **Quote** - Add blockquotes

**Publishing:**
- **Hero Image** - Upload a featured image
- **Published Date** - Choose when to publish (leave empty for draft)
- **Featured** - Check to show on homepage
- **Related Destinations** - Link to relevant destinations

4. Click **"Publish Post"** to make it live
5. Or **"Save as Draft"** to finish later

### Formatting Tips

**Headlines:** Use the heading dropdown for H2 or H3
**Bold Text:** Select text, click **B** button
**Add Link:** Select text, click link button, paste URL
**Add Image:** Click image button, paste image URL or upload

### Managing Existing Posts

- **Edit:** Click on any post in the list
- **Delete:** Click red trash icon
- **Search:** Use search box to find posts
- **Filter:** Show only published, drafts, or featured posts

## ðŸ–¼ï¸ Uploading Images

### Media Library

1. Click **"Media Library"** in sidebar
2. You'll see all uploaded images

### Uploading New Images

**Method 1: Drag & Drop**
1. Drag image file from your computer
2. Drop it on the upload zone
3. Upload starts automatically
4. Thumbnail appears when complete

**Method 2: File Picker**
1. Click **"Choose Files"** button
2. Select image(s) from your computer
3. Click **"Open"**
4. Upload starts automatically

**Image Requirements:**
- **Format:** JPG, PNG, WEBP, or GIF
- **Size:** Maximum 10MB per file
- **Dimensions:** At least 1920Ã—1280 recommended

### Using Images

**To use an image in a blog post:**
1. Upload image to Media Library
2. Click the image to open preview
3. Click **"Copy URL"** button
4. Paste URL into blog post editor

**To edit image details:**
1. Click image to open preview
2. Add **Alt Text** (describes image for accessibility)
3. Add **Caption** (optional)
4. Click **"Save Changes"**

### Deleting Images

1. Click image to open preview
2. Click **"Delete"** button (red)
3. Confirm deletion
4. Image is removed from storage

## âœï¸ Editing Website Copy

You can edit headlines, paragraphs, and button text directly on the website!

### How to Edit Static Text

1. Make sure you're logged in to the admin
2. Visit the main website homepage
3. Look for the **"Edit Mode: OFF"** button (bottom-right corner)
4. Click to turn it **ON** (button turns green)
5. All editable text now has blue borders
6. Click any text to edit it
7. Type your changes
8. Changed text gets an orange border
9. Click **"Save Changes"** button
10. Success! Your changes are live

### What Can You Edit?

**37 text elements across the site:**
- Hero headlines and subheadlines
- Section titles
- Descriptions
- Button text
- Footer tagline
- Copyright text
- Contact section text
- And more!

### Tips for Inline Editing

âœ… **DO:**
- Make small changes at a time
- Save frequently
- Preview on mobile after editing
- Keep text concise and impactful

âŒ **DON'T:**
- Make text too long (watch for layout issues)
- Use special characters that might break formatting
- Edit multiple pages at once without saving

### Keyboard Shortcuts

- **Ctrl + S** (Windows) or **Cmd + S** (Mac) - Save changes

## ðŸ†˜ Troubleshooting

### "I can't log in"

**Check:**
- Email and password are correct
- Caps Lock is OFF
- Try copying/pasting password
- Try different browser (Chrome, Firefox, Safari)
- Clear browser cache and try again

**Still stuck?** Contact your developer.

### "My changes aren't showing"

**Try:**
1. Wait 5 seconds and refresh page (F5)
2. Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
3. Clear browser cache
4. Try incognito/private window
5. Check if you actually clicked "Save" or "Publish"

### "Image won't upload"

**Check:**
- File is an image (JPG, PNG, WEBP, GIF)
- File is under 10MB
- Internet connection is working
- Try a different image
- Try a different browser

### "I deleted something by accident!"

**Don't panic:**
1. Contact your developer immediately
2. They can restore from database backup
3. Recent backups available (usually within 24 hours)

**Prevention:**
- Always confirm before deleting
- Start with drafts before publishing
- Test changes on one item first

### "Edit mode button doesn't appear"

**Check:**
- You're logged into the admin panel
- You're on the main website (not in /admin/)
- Refresh the page
- Clear browser cache
