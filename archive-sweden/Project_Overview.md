Project Overview

Luxury Travel Sweden is a sophisticated single-page website for a high-end travel service specializing in bespoke Swedish experiences.

Key Features:

Dynamic Hero Section with rotating Swedish landscape backgrounds
Interactive Map (Leaflet) with 15 destinations, filterable by season, theme, and category
LIV AI Concierge - An intelligent chatbot that guides users through creating custom itineraries
Six Journey Pillars - Thematic travel categories (Wellness, Design, Culture, Culinary, etc.)
Light/Dark Theme Toggle with localStorage persistence
Responsive Design - Mobile-first approach with fluid typography
Editorial Content - Stories, press coverage, Instagram feed
Comprehensive Footer with newsletter signup and rotating backgrounds

Technology Stack:

Vanilla HTML/CSS/JavaScript - No frameworks
Leaflet.js - Interactive maps
Lucide Icons - SVG icon library
Pexels API - Stock photography
CartoDB tiles - Map backgrounds

Design Approach:

Single-file architecture (~6000+ lines in index.html)
CSS custom properties for theming
Responsive using clamp() and media queries
Accessibility-first with ARIA attributes
Performance optimized with lazy loading
